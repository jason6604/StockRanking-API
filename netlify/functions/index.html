Stock Info API V1.1

API Document

API Method Example Description
getstockpricing Get 
http://{api-domain}/stockapi/getstockpricing/{stockid}

Get stocks pricing.

All data will be fetched if the stockId is not provided.
getstocksbyrank Get 
http://{api-domain}/stockapi/getstocksbyrank/{ranknum}

Get specified ranking stocks. (rank range: 3~20)

All data will be fetched if rank is not provided.
getcompanyinfo Get 
http://{api-domain}/stockapi/getcompanyinfo/{stockid}

Get company information.

All data will be fetched if the stockId is not provided.
getstockindecators Get 
http://{api-domain}/stockapi/getstockindecators/{stockid}

Get get stock indecators.

All data will be fetched if the stockId is not provided.
